# React Covid

React, Context/Hooks, Tailwind, TypeScript, modular, Single page app, Api, plugins, react-day-picker, Materials-UI.
React Website App where user can see the statistics of Covid virus across the world, he can choose a country to display more specific statistics or just see top 5 most infected countries. When he choose the country he gain access to module(react-day-picker) which displays the calendar where user can choose the day and see the statistics from specific day. Page has custom made animations. Page using simple routing.
